# Customer.io client for haskell

servant-client based client for track API of customer.io

Currently supported:
- [ ] Account Region
- [ ] Customers
- [x] Events
- [ ] Forms
- [ ] Segments

## Build Instructions

Run `nix develop -c cabal build` to build everything.
